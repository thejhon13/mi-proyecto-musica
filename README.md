# SoundCode Studio de jon villota

## Opción elegida

Opción D — Playground de música generativa con Astro y Strudel.

## ¿Por qué elegí esta opción?

Elegí esta opción porque combina desarrollo web moderno con música interactiva y permite experimentar con programación creativa.

---

## Descripción del proyecto

SoundCode Studio es una plataforma interactiva desarrollada con Astro.js y Strudel que permite explorar conceptos musicales como:

- Ritmo
- Melodía
- Armonía

Cada sección incluye patrones musicales precargados y un editor interactivo para reproducir sonidos en tiempo real.

---

## Tecnologías utilizadas

- Astro.js
- Strudel
- HTML
- CSS
- JavaScript
